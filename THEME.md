# Gather — UI Theme

Dark navy base with violet accents. Clean, modern chat app aesthetic.

## Colours

| Role             | Hex       | Usage                                      |
|------------------|-----------|--------------------------------------------|
| Background       | `#0F0F1A` | Page/app background                        |
| Surface          | `#1A1A2E` | Cards, panels, sidebar, message containers |
| Border           | `#2D2D44` | Dividers, input borders                    |
| Primary          | `#7C3AED` | Buttons, active states, highlights         |
| Primary Hover    | `#6D28D9` | Button hover                               |
| Primary Light    | `#A78BFA` | Links, secondary accents                   |
| Text Primary     | `#F8FAFC` | Main body text                             |
| Text Secondary   | `#94A3B8` | Timestamps, usernames, placeholder text    |
| Danger           | `#EF4444` | Errors, destructive actions                |

## Usage Notes

- Use **Background** for the outermost shell and page wrappers
- Use **Surface** for sidebar, message feed container, and form panels
- Use **Primary** for submit buttons and active room highlights
- Use **Text Secondary** for sender names and timestamps in the message feed
- Inputs should have a **Surface** background with a **Border** outline, white text
- Avoid pure black (`#000`) — use **Background** instead for depth
